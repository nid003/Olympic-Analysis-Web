# Olympic Analysis Web App

## Overview
An interactive Streamlit web application for analyzing Olympic Games data.

## Features
- Overall medal tally analysis
- Country-wise performance analysis
- Athlete-wise insights
- Participation trends over the years
- Interactive visualizations

## Technologies Used
- Python
- Pandas
- NumPy
- Streamlit
- Plotly
- Seaborn
- Matplotlib

## Dataset
- athlete_events.csv
- noc_regions.csv

## Project Structure
```
Olympic-Analysis-WebApp
│
├── app.py
├── helper.py
├── preprocessor.py
├── athlete_events.csv
├── noc_regions.csv
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
